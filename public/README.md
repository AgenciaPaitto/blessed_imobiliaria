# Pasta de Arquivos Públicos (Assets)

Coloque nesta pasta todos os arquivos estáticos que você deseja acessar diretamente no site, como logotipos, favicon, imagens de imóveis e documentos.

Qualquer arquivo colocado aqui será servido a partir da raiz do site.
Por exemplo:
- Se você salvar o logotipo como `logo.png` nesta pasta, poderá acessá-lo no código usando `/logo.png`.
- Se você salvar uma imagem de imóvel como `imovel1.jpg`, poderá acessá-la usando `/imovel1.jpg`.
